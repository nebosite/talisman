Stuff to know about Talisman:

Timers

Use the active timers tab to instantly create timers and manage existing timers.

Calendars

Talisman knows how to read your outlook calendar and automatically set up timers for the current day.

General

Talisman is a tool developer by Eric Jorgensen for himself. 

Talisman is open source.  You can leave comments and/or contribute via Github.
